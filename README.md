# Python-BlackJackBot

这是一个支持多人黑杰克的棋牌游戏，支持3.10+ 和python-telegram-bot 20+版本，

## 配置

用户可在`config.py`中配置自己的BOT TOKEN

### 1.) 安装和启动

``pip install -r requirements.txt``

Afterwards just run `python3 bot.py` and if done right, you'll be left with a working bot.
